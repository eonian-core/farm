# Subgraph for Eonian

This is Subgraph package for Eonian yield aggregator.

## First Start guide

1. Install dependencies

```bash
yarn
```

2. Install PostgreSQL. [Why it need?](https://thegraph.com/docs/en/developing/unit-testing-framework/).

```bash
sudo apt install postgresql
```

3. Test subgraph

```bash
yarn test
```
